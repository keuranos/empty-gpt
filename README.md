# Invoice Nudge Studio

A lightweight micro‑SaaS concept that helps freelancers and small agencies send payment reminders and recover cash flow. This repo ships a static MVP preview you can open in a browser.

## What it includes

- Landing page with positioning, pricing, and revenue plan.
- Reminder builder that generates a ready‑to‑send email.
- Local storage list of clients.
- Revenue calculator for €500–€1,000 MRR targets.

## Run locally

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/index.html`.
